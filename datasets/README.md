# Datasets

GLASS is trained on **DF2K** (DIV2K + Flickr2K) and evaluated on five standard
single-image super-resolution benchmarks: **Set5, Set14, BSD100 (B100),
Urban100, and Manga109**.

Download the datasets and arrange them under this directory as follows. The
paths match the defaults in the training/testing configs under `options/`.

```shell
|-- datasets
    # image SR - train
    |-- DF2K
        |-- HR
        |-- LR_bicubic
            |-- X2
            |-- X3
            |-- X4
    # image SR - test
    |-- test
        |-- Set5
            |-- HR
            |-- LR_bicubic
                |-- X2
                |-- X3
                |-- X4
        |-- Set14
            |-- HR
            |-- LR_bicubic
                |-- X2
                |-- X3
                |-- X4
        |-- B100
            |-- HR
            |-- LR_bicubic
                |-- X2
                |-- X3
                |-- X4
        |-- Urban100
            |-- HR
            |-- LR_bicubic
                |-- X2
                |-- X3
                |-- X4
        |-- Manga109
            |-- HR
            |-- LR_bicubic
                |-- X2
                |-- X3
                |-- X4
```

Notes:
- LR images are generated by bicubic downsampling. The configs expect filenames
  of the form `{name}x{scale}` (e.g. `0801x2.png`); Manga109 uses
  `{name}_LRBI_x{scale}`. Adjust `filename_tmpl` in the config if your files use
  a different convention.
- Common sources: DIV2K/Flickr2K from the NTIRE challenge, and the benchmark
  test sets from their respective project pages.
